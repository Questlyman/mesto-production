# Mesto Russia
Этот репозиторий содержит ТОЛЬКО production-сборку проекта.
Исходный код для разработки находится в другом приватном репозитории.

Проект Mesto - интерактивная страница для обмена фотографиями. Пользователь
может редактировать профиль и аватар, добавлять и удалять свои карточки,
ставить лайки, смотреть фотографии в полноэкранном режиме и открывать
статистику карточек.

Проект подключен к API-серверу Mesto и публикуется на GitHub Pages через
отдельный production-репозиторий.

## Функциональность

- Адаптивная верстка
- Загрузка профиля пользователя с сервера
- Загрузка карточек с сервера
- Редактирование профиля: имя и описание
- Обновление аватара пользователя
- Добавление новых фотографий
- Удаление только своих карточек
- Модальное окно подтверждения удаления карточки
- Постановка и снятие лайков
- Отображение количества лайков
- Просмотр фотографии в полноэкранном режиме
- Модальное окно со статистикой карточки
- Валидация форм
- Улучшенный UX форм с индикацией загрузки

## Запуск проекта

Установите зависимости:

```bash
npm install
```

Запустите локальный сервер разработки:

```bash
npm run dev
```

Соберите проект:

```bash
npm run build
```

Опубликуйте production-сборку в публичный репозиторий:

```bash
npm run deploy
```

## Деплой

Production-сборка публикуется в отдельный публичный репозиторий
`Questlyman/mesto-production`. Исходный код проекта хранится в этом
репозитории, а в публичный репозиторий отправляется только содержимое папки
`dist`.

Для автоматического деплоя через GitHub Actions в приватном репозитории должны
быть добавлены secrets:

- `PUBLIC_PAGES_TOKEN`
- `PUBLIC_PAGES_REPO`
